Description

Implemented Worm Motion.
Frameworks Used: OpenGL, GLUT functions Implemented:
  • To make each segment of the worm
  • Update the mass, direction of each segment with the help of springs and update the masses accordingly.

Key ideas for this project

  1. Segment Creation: Each segment of the worm has been meticulously crafted. This involves the design and rendering of individual components that collectively form the complete worm.
  2. Mass and Direction Update: The implementation incorporates dynamic behavior by updating the mass and direction of each worm segment. This dynamic aspect adds a lifelike quality to the animation.
  3. Spring-Based Mechanism: To achieve smooth and realistic motion, a spring-based mechanism is employed. This involves the use of springs to connect different segments of the worm, allowing for fluid movement and interaction between the segments.
